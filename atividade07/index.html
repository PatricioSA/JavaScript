Math.sqrt

Sintaxe:
Math.sqrt(x)

O método math.sqrt() retorna a raiz quadrada de um número

Exemplo:
let numeroMaior = Math.sqrt(20)

Math.max

Sintaxe:
Math.max(x, y)

O método math.max() retorna o número com maior valor

Exemplo:
let numeroMaior = Math.max(20, 10)

Math.min

Sintaxe:
Math.min(x, y)

O método math.min() retorna o número com menor valor

Exemplo:
let numeroMenor = Math.min(20, 10)

Math.pow

Sintaxe:
Math.pow(x, y)

O método math.pow() retorna o valor de x elevado a potência y

Exemplo:
let potencia = Math.pow(2, 4)

Math.round

Sintaxe:
Math.round(x)

O método math.round() arredonda um número para um inteiro mais próximo

Exemplo:
let arredonda = Math.round(2.75)

Math.trunc

Sintaxe:
Math.trunc(x)

O método math.trunc() retorna a parte inteira de um número
Nota: Este método não arredonda o número, ele apenas remove as partes decimais

Exemplo:
let parteInteira = Math.trunc(15.45)
